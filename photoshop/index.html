Photoshop Cheetsheet(단축기) 및 꿀팁 사이트

  • 새 파일 만들기 : Ctrl + N
  • 레이어 패널 : F7
  • 작업 취소하기 : Ctrl + Z
  • 이미지 불러오기 : Ctrl + O
  • 화면 크기 늘리기 : Ctrl + +
  • 화면 크기 줄이기 : Ctrl + -
  • 화면 크기 100%비율로 조정하기 : Ctrl + 1
  • 컴퓨터 사양 맞추는법 : F8 누르고 오른쪽 위에 설정 들어가서 Panel Options 들어가서 밑에 설정에서 Efficiency 클릭하기(100%뜨는거 확인하기)
  • 환경 설정 : Ctrl +
  • 브라켓 이미지 넣는법 : img src="이미지 넣은곳(저장한곳)" style="width:크기" alt="태그할꺼"
